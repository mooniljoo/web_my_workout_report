## Execute

'Node.js' required.

현재 디렉토리에서 아래 명령어를 입력한다.

```bash
yarn install
yarn start
```

개발모드로 웹앱이 실행되며<br />
[http://localhost:3000](http://localhost:3000)를 통해 웹앱을 볼 수 있다.

수정사항이 발생시 웹페이지는 자동새로고침된다.<br />
